                    Routing Options Explained
                

                
            

            
            
            

                
These settings control how the Generator (creates the shape) and the Router (finds the path) behave.

                
                

                    
                    

                        

                            
                            Prefer Separated Paths
                        

                        
Runner-focused separated-corridor mode. Prioritises traffic-separated links first, then paved footways, and can optionally fall back to quiet service lanes.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
Tier ladder used by this toggle

                                
    
                                    
  • Tier 1: highway=cycleway|path|pedestrian with foot=yes|designated|permissive -> 0.70x
    • 
                                    
  • Tier 2: highway=footway + footway=sidewalk + paved surface + foot=yes|designated|permissive -> 0.82x
    • 
                                    
  • Tier 3: highway=footway on paved surface -> 0.92x
    • 
                                    
  • Tier 4 (optional): quiet service-lane fallback when enabled -> 0.97x
    • 
                                

                                
Additional effects

                                
    
                                    
  • Major vehicle corridors highway=motorway|motorway_link|trunk|trunk_link|primary|primary_link|secondary|secondary_link|tertiary|tertiary_link -> 2.8x
    • 
                                    
  • Paved surface families on Tier 1-3 links -> extra 0.90x
    • 
                                    
  • Soft trail surfaces on Tier 1-3 links -> extra 1.25x
    • 
                                    
  • Public-right-of-way style hints in designation/public_footpath/prow on Tier 1 links -> extra 0.93x
    • 
                                

                            

                        

                    


                    

                        

                            
                            Prefer Segregated Paths
                        

                        
Adds a bonus-only preference for edges explicitly tagged as segregated. Missing segregated tags stay neutral.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
    
                                    
  • segregated=yes -> 0.90x bonus
    • 
                                    
  • segregated=no or missing -> 1.00x (neutral)
    • 
                                

                                
This toggle is enabled only when Prefer Separated Paths is active.

                            

                        

                    


                    

                        

                            
                            Allow Quiet Service Lanes
                        

                        
Allows a controlled fallback tier to selected service roads when stronger separated options are absent nearby.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
Quiet service eligibility

                                
    
                                    
  • highway=service
    • 
                                    
  • Parsed maxspeed exists and is <= 30 km/h (supports numeric and mph values)
    • 
                                    
  • Foot-friendly signal present: sidewalk=both|left|right|yes|separate or foot=yes|designated or bicycle=yes|designated
    • 
                                    
  • Soft trail surfaces are excluded from this fallback tier
    • 
                                

                                
Eligible lanes use Tier 4 multiplier 0.97x.

                            

                        

                    


                    

                        

                            
                            Prefer Nature Trails
                        

                        
Prefers trail-like paths and natural surfaces (where mapped), while strongly discouraging vehicle-heavy roads.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
Primary checks

                                
    
                                    
  • highway=path|track|bridleway|footway|steps -> 0.72× reward
    • 
                                    
  • surface=dirt|earth|ground|mud|sand|grass|grass_paver|woodchips|gravel|fine_gravel|compacted -> 0.78× reward
    • 
                                    
  • highway=motorway|motorway_link|trunk|trunk_link|primary|primary_link|secondary|secondary_link|tertiary|tertiary_link -> 4.0× penalty
    • 
                                    
  • highway=residential|unclassified|service|living_street -> 1.35× penalty
    • 
                                    
  • surface=paved|asphalt|concrete|concrete:plates|concrete:lanes|paving_stones -> 1.35× penalty
    • 
                                    
  • highway=path|track|bridleway|footway|steps with unknown surface -> extra 0.90× reward
    • 
                                

                                
When this is enabled, the solver disables Prefer Separated Paths and Prefer Paved Surfaces to avoid conflicting goals.

                            

                        

                    


                    

                        

                            
                            Prefer Paved Surfaces
                        

                        
Avoids unpaved surfaces (dirt, gravel, grass). Ideal for road cycling or keeping your shoes clean.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
Surface map used by the router

                                
    
                                    
  • surface=paved|asphalt|concrete|concrete:plates|concrete:lanes|paving_stones -> 1.0×
    • 
                                    
  • surface=sett|cobblestone|cobblestone:flattened|metal|wood -> 1.1×
    • 
                                    
  • surface=compacted|fine_gravel|gravel -> 1.3×
    • 
                                    
  • surface=dirt|earth|ground|mud|sand|grass|grass_paver|woodchips -> 2.0×
    • 
                                    
  • Missing or unknown surface -> 1.2×
    • 
                                

                                
This toggle only uses the OSM surface tag. It does not directly check traffic intensity or sidewalk presence.

                            

                        

                    


                    

                        

                            
                            Prefer Lit Streets
                        

                        
Applies lighting-aware bias using OSM and council-augmented edge lighting where available. Note: Sparse OSM data or missing local council data can reduce effect strength.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
Lighting classes and multipliers

                                
    
                                    
  • lit=yes|automatic|24/7 -> class lit -> 0.85×
    • 
                                    
  • lit=limited|disused -> class limited -> 1.3×
    • 
                                    
  • lit=no -> class unlit -> 1.8×
    • 
                                    
  • Missing/unknown class -> default 1.2×, except dedicated paths with unknown lighting use 1.0×
    • 
                                

                                
Other tags that affect class resolution

                                
    
                                    
  • lighting_regime all-night hints: all night, all_night, allnight, 24/7, 24h, 24 hour, dusk to dawn, dusk-dawn, sunset to sunrise
    • 
                                    
  • lighting_regime part-night hints: part night, part_night, partnight, switch off, switch_off, switchoff, midnight, curfew
    • 
                                    
  • lighting_context: daylight disables lighting multipliers, twilight/night enables them
    • 
                                

                            

                        

                    


                    

                        

                            
                            Heavily Avoid Unlit Streets
                        

                        
Much stronger version of "Prefer lit streets". Confirmed unlit streets receive a 5× cost penalty; unknown street lighting receives a 3× penalty. Unknown lighting on dedicated paths is treated as neutral. Note: Sparse OSM lighting data, or missing/non-overlapping council lighting data, can produce longer detours.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
Heavy lighting table

                                
    
                                    
  • lit -> 0.70×
    • 
                                    
  • limited -> 2.5×
    • 
                                    
  • unlit -> 5.0×
    • 
                                    
  • unknown -> default 3.0×, except dedicated paths with unknown lighting use 1.0×
    • 
                                

                                
This mode uses the same OSM tag parsing as Prefer Lit Streets (lit, lighting_regime, highway, and lighting_context) but applies stronger penalties.

                                
Selecting this option programmatically disables Prefer Lit Streets in the solver to keep behavior deterministic.

                            

                        

                    


                    

                        

                            
                            Avoid Unsafe Roads
                        

                        
Heavily penalises major roads (Highways, Trunk roads, busy Primaries). The router will actively avoid these.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
OSM tags checked

                                
    
                                    
  • highway=primary|primary_link|secondary|secondary_link|tertiary|tertiary_link marks an edge as potentially unsafe
    • 
                                    
  • Safe sidewalk values: sidewalk=both|left|right|yes|separate
    • 
                                    
  • Safe foot values: foot=yes|designated
    • 
                                    
  • If an edge is in an unsafe highway class and has no safe sidewalk/foot signal -> 3.5× penalty
    • 
                                

                                
Shared active-travel quality bonus while enabled

                                
When a dedicated hard-surface path has strong pedestrian/cycle designation, an extra bonus can apply: tier_a 0.82×, tier_b 0.90×, tier_c 0.96×.

                            

                        

                    


                    

                        

                            
                            Avoid Unclassified Country Lanes (Last Resort)
                        

                        
Strongly penalises unclassified roads that lack pedestrian/cycle safety cues. This behaves like a soft ban: avoided whenever alternatives exist, but still available as a fallback when necessary.

                        

                            
                                Show advanced (OSM tags and multipliers)
                            
                            

                                
When the penalty applies

                                
    
                                    
  • highway=unclassified
    • 
                                    
  • No safe foot markers: sidewalk=both|left|right|yes|separate or foot=yes|designated
    • 
                                    
  • No safe cycleway markers: cycleway or cycleway:both in lane|track|separate|yes|shared_lane|share_busway|opposite_lane|opposite_track
    • 
                                    
  • Applied multiplier: 8.0×
    • 
                                

                            

                        

                    


                    
                    

                        

                            
                            Scenic Bearings
                        

                        
Influences Loop Shape. Instead of choosing random directions, the solver scans the map for Green/Water areas and orients the loop towards them. 
Disabling this reverts to fixed triangle shapes.

                        

                            
                                Show advanced
                            
                            

                                
This option changes loop waypoint orientation before final A* routing. It does not add direct per-edge OSM tag multipliers in the same way as the routing modifiers above.

                            

                        

                    


                    

                        

                            
                            Route Variety
                        

                        
Influences Loop Shape. 

                        Low: Simple Triangles (Predictable). 

                        High: Complex Shapes (Quads/Pentagons) with random variation. Generates more natural-looking routes.

                        

                            
                                Show advanced
                            
                            

                                
This option controls loop complexity/randomness in shape generation. It does not directly alter edge multipliers tied to OSM tags during WSM A* edge scoring.
